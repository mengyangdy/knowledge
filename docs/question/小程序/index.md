## 1.2 背景
⼩程序并⾮凭空冒出来的⼀个概念，当微信中的 WebView 逐渐成为移动 Web 的⼀个重要⼊⼝时，微信就有相关的 JS-SDK

## 1.3 优缺点

- 流量⼤，易接受：⼩程序借助⾃⾝平台更加容易引⼊更多的流量
- 安全
- 开发⻔槛低

- 受控微信：⽐起APP，尤其是安卓版的⾼⾃由度，⼩程序要⾯对很多来⾃微信的限制，从功能接⼝，甚⾄到类别内容，都要接受微信的管控
